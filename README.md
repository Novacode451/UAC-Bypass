    press 2 and enter your targets username
    enter your password list (passlist.txt for default passlist)
    wait

NOTE: if it stops at a random attempt that means the password its on contains one of these characters: !><|^ (or another weird character), just delete that attempt from the file and restart
Fake UAC Prompt

    close any cmd windows you got open
    run main.vbs
    hide the folder (to not be sus)
    ask parent/teacher to help
    enjoy the admin cmd
